## How CAT Score-to-Percentile Conversion Works
The score-to-percentile system is influenced by several factors:
- Exam slot difficulty  
- Normalization rules applied by IIMs  
- Scaled scores in each section  
- Candidate performance distribution  
Although the exact scaling process changes every year, advanced percentile predictors use past year trends, difficulty levels, and mock test data to estimate percentile with high accuracy.
